#### 다운로드, 환경설정  
ubuntu 16.04 LTS 기준  

```
apt install golang-go
```

아무 폴더에서 테스트 파일을 작성해 본다.
- test.go
```
package main
func main(){
	println("Hello Go")
}
```

터미널에서 go run 명령어를 실행하면 컴파일과 동시에 실행된다.  
```
go run test.go
```

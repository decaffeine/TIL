### tar, tar.gz

http://sarghis.com/blog/468/  

- tar : 아카이브 파일 (여러 파일을 한 파일로 묶은 것)  
- tar.gz : tar로 묶인 아카이브를 gzip으로 압축한 것  

- 명령어 
  - 압축하기 tar -cvfz  
  - 압축풀기 tar -xvfz  

- 옵션 값  
  - c : 파일을 묶음  
  - v : 묶음/해제 과정을 화면에 표시  
  - z : gnuzip을 사용  
  - f : 파일 이름을 지정  
  - x : 묶음을 해제  

- gzip : GNU zip의 준말
